# Vocational Training Development Institute
his project is an assignment for the Vocational Training Development Institute,  
School of Applied Technology, Department of ICT. It focuses on building a **Java Swing GUI Payroll Application**, 
applying Object-Oriented Programming principles such as inheritance, interfaces, exception handling, and file persistence.
The project demonstrates how to design a professional payroll system with multiple employee types,
contractor invoices, and modern data handling.
